## 关于static

发现Private Functions(就是那些用static的)**不能**放在头文件里，否则就会要求在**每个**包含这个头文件的.c文件里都给出自己的定义，没有定义的话，这函数在该文件里就相当于*空函数*，这显然不是我们想要的。

虽然用OOP范式来写这玩意不会用到函数本身，但是make一下出一坨warning也是很难受的嘛...如果不是试了试这么写就发现不了这个了...
